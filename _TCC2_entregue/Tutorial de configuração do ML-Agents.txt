Para conseguir alterar e executar o projeto será necessário ter instalados os seguintes programas:
- Unity (foi utilizada a versão 2018.3.8f1)
- Anaconda (foi utilizada a versão 2019.10)

E além disso é necessário o download da biblioteca Unity ML-Agents (foi utilizada a versão 0.13.1).

O download desses programas pode ser feito em seus sites oficiais.
Instruções para a instalação e configuração do ML-Agents se encontram no Apêndice C do artigo.

Caso ainda existam dúvidas, é possível utilizar o tutorial desenvolvido pela própria equipe do Unity, no link:
https://learn.unity.com/project/ml-agents-penguins

Nele são explicados todos os passos para instalação e configuração dos recursos necessários para poder realizar o treinamento com o ML-Agents utilizando um projeto de exemplo.
